# .brain

This folder contains the project guidance documentation used by Codex CLI, Codex Web, Claude Code, and Antigravity.

Recommended read order:
1. `PROJECT_OVERVIEW.md`
2. `ARCHITECTURE.md`
3. `PRODUCT_SPEC.md`
4. `BUILD_ORDER.md`
5. `CURRENT_FOCUS.md`
6. `TODO_BACKLOG.md`
7. `DECISIONS_LOG.md`
8. `CODING_GUIDELINES.md`
9. `AGENT_README.md`

Operational helper:
- `SAFE_GITHUB_UPDATE.md` for merging only `.brain` (and optionally the bootstrap script) into `main` without touching other files.

Additional operator docs:
- `LOCAL_RETRIEVAL_STEPS.md` for pulling from GitHub and bootstrapping locally on Windows.
